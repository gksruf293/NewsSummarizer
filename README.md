## 🚀 왜 AI News English Study인가요?
뉴스를 읽으며 영어 공부를 하고 싶지만, 너무 어려운 단어와 복잡한 문장에 좌절하신 적 없나요? 
본 서비스는 **GPT-4o-mini**를 활용하여 동일한 뉴스를 **초등/중등/고등** 3단계 수준으로 맞춤 요약해 드립니다.

### ✨ 핵심 포인트
- **📊 단계별 학습 (3-5-7 Rule)**: 
  - **Level 1**: 기초 단어로 이루어진 짧은 3문장 요약
  - **Level 2**: 중급 어휘를 포함한 5문장 요약
  - **Level 3**: 고급 어휘와 복잡한 구문의 7문장 요약
- **🔍 지능형 시맨틱 검색**: 단순 키워드가 아닌 '의미'를 이해하는 AI 검색 기능
- **🖱️ 클릭 번역**: 영문장을 클릭하면 숨겨진 한국어 해석이 즉시 노출
- **🖼️ 깔끔한 UI**: 이미지가 없는 뉴스도 보기 좋게 자동 정돈
